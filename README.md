## Kaynak kodundan calistirma
1. Oncelikle bir venv olusturun ve giris yapin (eger kullanilacak kutuphaneleri tum sistem icin yuklemek isterseniz bu asamayi gecebilirsiniz):
   python -m venv venv
   
   source venv/bin/activate (linux)
   
   venv\scripts\activate (windows)
   
3. Gerekli kutuphaneleri yukleyin:
   pip install -r requirements.txt

4. Python kodunu calistirin:
   python src/main.py
